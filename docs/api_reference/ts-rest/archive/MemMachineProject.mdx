## Class `MemMachineProject`

The `MemMachineProject` class provides methods for managing the lifecycle and configuration of a specific project workspace.

### Methods

### `create()`
Initializes the project on the MemMachine server with specific AI model configurations.

```typescript
await project.create(options: CreateProjectOptions)
```

**Parameters**

- `options`: [`CreateProjectOptions`](../interfaces/ProjectContext) — Settings for embedders and rerankers.

### `getDetails()`

Retrieves the full metadata and current state of the project.

```typescript
await project.getDetails()
```

**Returns**

- `Promise<Project>` — The [Project](../interfaces/Project) data model.

<Note title="Direct Access"> While this class provides project-level utilities, most memory operations (adding and searching) are now performed directly through the [MemMachineClient](../classes/MemMachineClient) using metadata scoping. </Note>
