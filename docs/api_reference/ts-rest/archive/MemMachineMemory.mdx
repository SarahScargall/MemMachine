## Class `MemMachineMemory`

This class handles operations related to specific memory entries.

### Methods

### `delete()`
Permanently removes a specific memory entry from the project.

```typescript
await memory.delete(memoryId: string)
```

**Parameters**

- `memoryId`: The unique identifier of the memory to be deleted.

### `update()`

Updates the content or metadata of an existing memory.

```typescript
await memory.update(memoryId: string, content: string)
```

<Tip title="Standard Workflow"> For high-volume ingestion and semantic retrieval, use addEpisodicMemory and searchProfileMemory via the [MemMachineClient](../classes/MemMachineClient). </Tip>
