## Playground Overview

The MemMachine **Playground Dashboard** serves as your central hub for monitoring usage, accessing features, and managing your account. Think of it as your console's home screen.

### Usage Monitoring (Graphs)

This section displays key metrics related to your **Playground** activity, including API call volume, processing time, and resource consumption. You'll see this area prominently featured: 

Since this is a new account, the graphs may initially appear empty, but don't worry!

* **Tip:** Data will populate these graphs as soon as you begin making API calls using your generated API keys.

## Navigation and Controls

The **Playground** provides two main navigation areas: the **Left-Hand Menu** for primary feature access and the **Top Menu Bar** for account-specific actions.

### Left-Hand Menu (Primary Features)

Use this menu to quickly navigate between core areas of the **Playground**:

| Link Name | Description |
| :--- | :--- |
| **Dashboard** | Returns you to this main overview screen. |
| **Playground** | Access the interactive environment for testing tools and running queries. |
| **Documentation** | Links to the full MemMachine documentation suite. |
| **API Keys** | Where you can create, view, and manage your API access tokens. |
| **Billing** | Access information regarding your subscription and usage costs. |

### Top Menu Bar (Account Actions)

The top-right corner of the dashboard provides quick access to essential account features, detailed below:

<Accordion title="Notifications (Bell Icon)">
Clicking the **Bell Icon** reveals a panel that shows recent system updates, maintenance alerts, and messages specific to your account activity. This is where you'll find important system communications. 
</Accordion>

<Accordion title="Account and Settings (User Icon)">
Clicking the **User Icon** opens a drop-down menu with several account options, including:
* **Profile:** Edit your personal information and preferences.
* **Settings:** Configure general **Playground** settings.
* **Logout:** Securely sign out of the MemMachine **Playground**.

</Accordion>
